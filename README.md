## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/lucas-canellas/dac-2023.1.git
```

Entre no diretório do projeto

```bash
  cd dac-2023.1
```



Inicie o servidor

```bash
  mvn spring-boot:run
```

Acesse a URL

```bash
  http://localhost:8081
```
